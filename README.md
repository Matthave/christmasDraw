This is one of first application which I made.

Technologies that have been used:
Java Script with Object-Oriented, CSS3, HTML5

What is this application?
An application with which you can determine how many people are to take part in the christmas draw. And determine who to give the present to whom.
It is not possible for the same people to be drawn at random or for the draw not to succeed

Download Repo
If you want to download my code, you must clone the repository with the command 
git clone https://https://github.com/Matthave/christmasDraw

Launching the application
After cloning the application you must use the command npm install
 
 Next: You can run this application with Live Server

You can also see this page at this address www.christmasdraw.pl
